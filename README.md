Sea battle
===============================

## Required

 - node >= 4.8.1

## Setup

- Install project dependencies
```text
$ npm install
```
- Run app
```text
$ npm run serve 
```
- To build production 
```text
$ npm run build
```
- To run tests
```text
$ npm run test
```
